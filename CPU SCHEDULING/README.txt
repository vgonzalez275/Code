To compile Program: 

           gcc -g -o proj2 cpu.c

To run program: 

           gdb proj2

In gdb type command: 
         
          run proj2 input.1 FCFS           
          run proj2 input.1 SRTF or SJF
          run proj2 input.1 RR 1


Note: It doesnt run regularl because it causes segmentation fault. 
             It only runs and works in debugger.
